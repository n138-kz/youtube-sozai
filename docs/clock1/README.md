# youtube-sozai

## clock1

### How to use

#### OBS

1. `ソース` -> `追加` -> `ブラウザ`
2. `新規作成`, 名前: ブラウザ (任意の重複しない名前)
3. 設定項目は下記表参照
    | _ | _ |
    | :- | :- |
    | URL | `https://n138-kz.github.io/youtube-sozai/clock1` |
    | 幅 | `500` |
    | 高さ | `250` |

### Customize

#### 文字色
```css
:root, html, body {
    --text-color: azure;
}
```
#### フォント
```css
:root, html, body {
    font-family: uzura_font;
}
```
#### 影文字
```css
:root, html, body {
    --text-shadow-size: 8px;
    --text-shadow-color: #333;
}
```
